나의 Alidata 인턴십 경험

회사 소개

Alidata는 1984년에 설립된 포르투갈 회사로, 경영 소프트웨어 개발을 전문으로 합니다. 40년 넘는 기간 동안 혁신 능력과 고객 요구에 대한 지속적인 적응, 그리고 시장 변화에 발맞춘 IT 솔루션 개발로 기술 분야에서 두각을 나타내고 있습니다. 현재 앙골라, 브라질, 중국(마카오) 등 여러 대륙에 회사를 두고 있습니다.

Alidata에는 하드웨어와 소프트웨어 두 가지 주요 부서가 있습니다. 하드웨어 부서에서는 컴퓨터 조립, 고객 및 파트너와 기기 문제 해결, 그리고 같은 층에 프린터 관련 부서가 있습니다.

소프트웨어 부서는 제가 있었던 개발 부서로, 프로그래머들이 고객 요청에 맞춘 공동 작업을 수행합니다. 또한 업무 조직 및 배분 부서, 교육 부서, 기술 지원 부서, 마케팅 부서가 있습니다.


수행한 활동

인턴십 동안 SQL 프로그래밍을 배웠고, FireBird 데이터베이스(IBExpert IDE)를 사용했습니다. Visual Studio Community와 C#을 통해 데이터베이스에 연결하여 데이터 기록과 수정 작업을 했습니다. 이 프로젝트가 끝난 후 HTML5와 CSS3를 독학하여 작은 웹사이트 프로젝트를 완성했습니다.

또한 하루는 하드웨어 부서에서 근무하며 서버를 구성하고, TPM 2.0 우회 방법을 사용해 Windows 11이 설치된 가상 머신을 세팅해 인터넷 없이 로컬 모드로 사용하는 경험을 했습니다.


겪었던 어려움

처음에는 새로운 데이터베이스와 SQL 프로그래밍, Visual Studio 연결에 적응하는 데 어려움이 있었지만, 시간이 지날수록 점점 더 이해하기 쉽고 편해졌습니다.


중요한 점

직원들과의 관계는 언제나 전문적이었고, Alidata 팀은 활기차고 긍정적이며 친근했습니다. 사람들과 어울리기 편했고 모두와 잘 지냈습니다.

인턴십 멘토는 제 작업에 항상 관심을 기울였고 잘 진행되는지 확인하며 도와주었지만, 제가 자율적으로 일할 수 있도록 충분한 자유를 주었습니다. 5주간 인턴 기간 동안 자율성과 독립성을 키울 수 있었습니다.


배운 점과 더 배우고 싶은 점

Firebird라는 조금 더 전문적이고 고급 데이터베이스를 사용하는 법을 배웠으며, SQL 프로그래밍도 익혔습니다.

C#과 SQL을 혼합해 데이터베이스에 연결하는 것도 배웠습니다. 도전적이었지만 노력과 헌신으로 해낼 수 있었습니다.

웹 개발에 가장 흔히 쓰이는 HTML과 CSS도 독학해 더 많은 자율성과 학습 의지를 얻었습니다.

짧은 기간 탓에 배우지 못한 점은 자바스크립트 프로그래밍 언어인데, 웹 페이지 개발에 매우 유용한 기술입니다.


결론

짧은 시간 동안 새로운 기술과 능력을 개발할 수 있었고, 과제를 수행하며 자율성과 독립성, 결단력을 키웠다고 생각합니다.

이 경험은 학교 환경에서는 얻기 어려운 실제 업무 환경에서의 경험이었으며, 도움을 주신 모든 분들, 특히 마지막까지 큰 도움을 준 넬리와 페드로에게 감사드립니다. 더할 나위 없는 인턴십 경험이었습니다.

고마워요, Alidata!
